class ContextImport(module, extra_dir=None)[source]¶

Bases: object

Import module context manager. Temporarily prepends extra dir to sys.path and imports the module,

Example:
>>> # /path/dir/fi.py
>>> with ContextImport('/path/dir/fi.py') as module:
>>>     # prepends '/path/dir' to sys.path
>>>     # module = import_module('fi')
>>>     module.main()
>>> with ContextImport('dir.fi', '/path') as module:
>>>     # prepends '/path' to sys.path
>>>     # module = import_module('dir.fi')
>>>     module.main()
